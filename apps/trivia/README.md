# Trivia Night

Team trivia game with presenter and audience views, dual-screen sync, and
accessibility-first UI.

## Key Routes

- `/play` - Presenter view
- `/display` - Audience view

## Local Development

```bash
pnpm dev:trivia
```

## Environment

See `.env.example` for required variables.

## Testing

```bash
pnpm test:run
```

- Audience components: [`src/components/audience`](./src/components/audience)
