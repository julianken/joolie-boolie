# Platform Hub Components

## Subdirectories

- `auth/`: Login, signup, and password reset forms.
- `dashboard/`: Dashboard cards and widgets.
- `providers/`: App-level providers and error boundaries.

## Conventions

- Keep layout components lean and prefer composition.
- Export public components from `components/index.ts`.
