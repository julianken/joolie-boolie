# Platform Hub

Central landing experience for selecting games and managing auth flows.

## Key Routes

- `/` - Landing page
- `/login` - Login
- `/signup` - Sign up
- `/forgot-password` - Reset password
- `/dashboard` - User dashboard

## Local Development

```bash
pnpm dev:hub
```

## Environment

See `.env.example` for required variables.

## Testing

```bash
pnpm test:run
```
