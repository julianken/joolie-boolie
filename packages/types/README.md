# @beak-gaming/types

Shared TypeScript types for the Beak Gaming Platform. Use this package to
avoid cross-app type drift.

## Installation

```json
{
  "dependencies": {
    "@beak-gaming/types": "workspace:*"
  }
}
```

## Public API

All exports are defined in `src/index.ts`.

## Usage

```typescript
import type { GameSession } from '@beak-gaming/types';
```
